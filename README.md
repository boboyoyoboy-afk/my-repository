# Monthly Days Off Calendar

A lightweight browser-based calendar for tracking accrued monthly days off, holidays, family occasions, and a custom event type.

## Features

- Monthly calendar with inline event pills and one-click delete actions.
- Days off bank that accrues by **+1 at the start of each new month after your work start date**.
- Date modal with checkboxes for day off, national holiday, family occasion, and a customizable event label.
- Settings modal for color customization and a semi-transparent background image upload.
- Local storage persistence so your planner remains available between refreshes.

## Run locally

Open `index.html` directly in a browser, or start a simple static server:

```bash
python3 -m http.server 4173
```

Then visit <http://localhost:4173>.
